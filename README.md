# RaspberryPi4-Temperature-DHT22
Mesure de la température en utilisant Raspberry pi 4

# Objectif du projet
- Mesurer la température et l’humidité ambiante avec précision.
-Afficher les données en temps réel sur un écran LCD.
-Fournir un système simple pour la surveillance environnementale.
-Enregistrer et visualiser les données de manière pratique.

# Contenu
-RaspberryPi4-Temperature-DHT22.pdf – Documentation complète du projet.
-Cablage-DHT22-LCD.png – Schéma du câblage du capteur DHT22 avec le Raspberry Pi et le LCD.

# Technologies
-Raspberry Pi 4
-Capteur DHT22
-Écran LCD
-Langage Python
